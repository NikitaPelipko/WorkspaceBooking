# WorkSpace — Frontend

Простой фронтенд для системы бронирования рабочих мест в университете.
**Стек:** Vanilla JS + Bootstrap-совместимый собственный CSS (без зависимостей).

---

## Структура

```
workspace-booking/
├── index.html              ← редирект на auth.html
├── css/
│   └── style.css           ← все стили (дизайн-токены + компоненты)
├── js/
│   └── api.js              ← API-клиент, Auth, утилиты UI
└── pages/
    ├── auth.html           ← Страница входа / регистрации
    ├── home.html           ← Главная (статистика + ближайшие брони)
    ├── buildings.html      ← Выбор корпуса
    ├── floors.html         ← Выбор этажа
    ├── floor-detail.html   ← Просмотр помещений + форма бронирования
    └── profile.html        ← Личный кабинет + история броней
```

---

## Настройка

### 1. API URL

В файле `js/api.js` измените:

```js
const API_BASE = 'http://localhost:8080/api';
```

на адрес вашего бэкенда.

### 2. CORS

Убедитесь, что Spring-бэкенд разрешает запросы с домена фронтенда.

### 3. Запуск

Любой статический сервер, например:

```bash
# Python
python3 -m http.server 3000

# Node.js (npx)
npx serve .

# VS Code: Live Server
```

Откройте `http://localhost:3000`

## API Endpoints (из бэкенда)

| Метод  | Путь                                              | Описание                        |
|--------|---------------------------------------------------|---------------------------------|
| POST   | /api/auth/register                                | Регистрация                     |
| POST   | /api/auth/login                                   | Вход                            |
| POST   | /api/auth/refresh                                 | Обновление токена               |
| GET    | /api/buildings                                    | Список корпусов                 |
| GET    | /api/buildings/{id}/floors                        | Этажи корпуса                   |
| GET    | /api/map/floors/{floorId}?from=...&to=...         | Карта этажа с доступностью      |
| GET    | /api/floors/{floorId}/rooms                       | Комнаты этажа                   |
| GET    | /api/rooms/{roomId}/workplaces                    | Рабочие места в комнате         |
| POST   | /api/booking/                                     | Создать бронь                   |
| GET    | /api/booking/my                                   | Мои брони                       |
| DELETE | /api/booking/{id}                                 | Отменить бронь                  |

## Ключевые особенности

- **JWT авторизация** с автоматическим refresh токеном
- **FloorMap API** — используется для отображения доступности в реальном времени
- **Бронирование двух типов:** `ROOM` (вся комната) и `WORKPLACE` (рабочее место)
- **Фильтрация по времени** на странице выбора этажа / помещения
- **Автоматическое обновление** доступности после бронирования
- **Адаптивный дизайн** — работает на мобильных устройствах
